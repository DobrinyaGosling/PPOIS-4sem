# lab_2 (вариант 5)

Оконное приложение (MVC) для ведения массива записей пациента:

- ФИО пациента
- Адрес прописки
- Дата рождения
- Дата приёма (последнего)
- ФИО врача
- Заключение

Функции: добавление, поиск (в отдельном диалоге с выводом результатов), удаление по условиям (в отдельном диалоге), постраничный вывод, альтернативный вывод в виде дерева, хранение в PostgreSQL, сохранение/загрузка в XML (DOM для записи, SAX для чтения).

## Запуск

1) Установить зависимости:

```bash
pip install -r requirements.txt
```

2) Запустить:

```bash
python -m src.main
pytest -q --cov=src --cov-report=term-missing

```

## База данных

PostgreSQL поднимается через `docker-compose.yaml`. Параметры подключения берутся из `.env` (см. `src/settings.py:1`).

```bash
docker compose up -d
```

Состояние приложения (размер страницы) хранится в `src/data/state.json`.
